# WiCaliper 复现工程文档

## 1. 复现总架构

```text
WiCaliper 复现总架构
│
├── 模块零：背景信号估计与 CSI 预处理          [对应原论文 §V Signal Processing]
│   ├── 采集无物体时的 CSI 基线序列
│   ├── 计算背景均值 H_bg = mean(H_baseline)
│   └── 输出目标差分信号 H_ob = H_rx - H_bg
│
├── 模块一：正向物理仿真引擎                  [对应原论文 §III 公式(13)~(17)]
│   ├── 从介电常数 ε_r 推导 α、β（麦克斯韦方程组）
│   ├── 计算透射系数 T_io（双界面折射率模型）
│   ├── 构建截面厚度函数 d(x)（圆柱/长方体/圆管）
│   ├── 计算穿透因子 P_p(x)（公式13，向量化）
│   ├── 数值积分计算衍射因子 P_d(w, h)（公式17）
│   │   └── 注：ρ_0 须由 Tx-Rx 几何参数自动派生
│   └── 合并输出属性函数 P(x) = P_p(x) · P_d(w, h)
│
├── 模块二：多视角传递矩阵构建                [对应原论文 §IV-A 公式(20)]
│   ├── 对每个视角 ψ（对应每对 Tx-Rx 天线）：
│   │   ├── 在视角坐标系下离散化物体 x 轴（N 个点）
│   │   ├── 对每个 Rx 采样位置计算 d_tx、d_rx
│   │   ├── 完整实现 f(p)：含斜率因子 K_θ 和 1/(r_T·r_R) 衰减
│   │   └── 构建 M×N 传递矩阵 F_ψ（M ≥ N）
│   └── 输出多视角矩阵集合 {F_ψ1, F_ψ2, ...}
│
├── 模块三：截断 SVD 属性函数恢复             [对应原论文 §IV-A 公式(21)]
│   ├── 对每个视角 ψ，输入 (F_ψ, H_ob_ψ)：
│   │   ├── 对 F_ψ 进行全量 SVD 分解：F = U Σ V*
│   │   ├── 以阈值=10 截断微小奇异值（置零）
│   │   ├── 计算截断伪逆：F̃⁺ = Ṽ Σ̃⁺ Ũ*
│   │   └── 求解 P̂_ψ = F̃⁺ · H_ob_ψ
│   ├── 闭环验证：对比 P̂_ψ 与 P_true 的幅度和相位曲线
│   └── 输出多视角估计集合 {P̂_ψ1, P̂_ψ2, ...}
│
└── 模块四：贝叶斯联合参数寻优                [对应原论文 §IV-B Algorithm 1]
    ├── 输入：多视角 {P̂_ψ}、视角角度集合 {ψ}、形状类型 i
    ├── 定义目标函数 L(ε_r, a, w, h)：
    │   ├── 由 (ε_r, a, w, h) 经模块一生成模拟属性函数 P̃_ψ
    │   └── 计算多视角总损失 L = Σ_ψ ‖P̃_ψ - P̂_ψ‖
    ├── 调用贝叶斯优化器（scikit-optimize 或 BayesianOptimization 库）
    │   ├── 搜索空间：ε_r ∈ [1, 81], 尺寸参数 ∈ [1cm, 20cm]
    │   ├── 默认迭代次数 K = 100
    │   └── 采集函数驱动下一采样点选择
    └── 输出最终估计：(ε_r*, a*, w*, h*)
```

## 2. 模块间的数据流与闭环验证

```text
真实/仿真数据
   │
[模块零] H_ob = H_rx - H_bg            ← 仿真时可跳过，直接输出 H_clean
   │
   ├────────────────────────┐
   │ (同时进行)             │
   ▼                        ▼
[模块二] F_ψ1, F_ψ2, ...   [模块一] P_true (仅验证时使用)
   │                        │
   ▼                        │
[模块三] P̂_ψ1, P̂_ψ2, ...   │
   │                        │
   └──── 闭环对比 P_true ───┘ (验证模块一+二+三的正确性)
   │
   ▼
[模块四] → ε_r*, a*, w*, h*
```

## 3. 关键工程约束清单

| 约束项 | 具体要求 | 来源 |
| :--- | :--- | :--- |
| **超定方程** | 测量位置数 $M \ge N$ (离散化点数) | 原论文 §IV-A |
| **空间步长** | $\Delta x = 2\text{ mm}$ | 原论文 §V |
| **SVD 截断阈值** | 有效奇异值阈值设为 $10$ | 原论文 §VI-A |
| **迭代次数** | $K = 100$ (精度与速度平衡点) | 原论文 §VI-E |
| **搜索范围** | $\epsilon_r \in [1, 81]$，尺寸 $\in [1, 20]\text{ cm}$ | 原论文 §VI-A |
| **传递矩阵** | 须包含 $K_\theta$ 斜率因子和 $1/(r_T \cdot r_R)$ 衰减 | 原论文公式(5) |
| **多视角最少数量** | 至少 $2$ 个独立视角（对应 Rx1 和 Rx2） | 原论文 Algorithm 1 |
| **$\rho_0$ 计算** | 必须由实际 Tx-Rx 距离派生，不可硬编码 | 原论文公式(17) |
